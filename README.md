# AQI Search App

A simple Air Quality Index (AQI) Search Engine built with Node.js (backend) and React.js (frontend).  
It allows users to search for a city and view detailed air quality information, including AQI, dominant pollutant, and individual pollutants.

Features

1.Search for any city by name.
2. Displays AQI with color-coded indicator
3. Shows dominant pollutant.
4. Displays detailed pollutant levels in a clean grid layout.
5. Responsive and interactive UI.
6.Backend caches API responses for faster repeat queries.

Run backend:
node server.js
Run frontend:
open terminal from frontend
npm install
npm run dev
